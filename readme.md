# Datingsite

This is my datingapp.

## Installation

To install this repository, type: 

```
Git clone https://github.com/emmaoudmaijer/datingsite
```
To install Node modules, type:
```
npm install
```
To run the application, type
```
npm run start
```


## Required Modules

* Camelcase
